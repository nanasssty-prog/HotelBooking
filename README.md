Ссылка на репозиторий GitHub: https://github.com/nanasssty-prog/HotelBooking

## Технологии:

- **Frontend:** HTML5, CSS3, JavaScript.
- **Инструменты:** VS Code, Git, Node.js, npm.
- **Библиотеки и зависимости:** lodash, nodemon.
- **Дизайн:** Figma.

## Расширения в VS Code

- Live Server
- Auto Rename Tag
- Prettier
- Code formatter
- GitLens
- ES6 String HTML
- Figma to Code

## Как запустить проект

1. Склонируйте репозиторий.
2. Установите зависимости командой `npm install`.
3. Откройте `index.html` через Live Server в VS Code.
